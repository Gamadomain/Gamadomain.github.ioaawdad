# Gamestore Website - Монгол Алт Худалдаа Оо

Энэ вебсайт нь үйлчүүлэгчдэд алт худалдаж авахад шаардлагатай мэдээлэл бүрдүүлэх боломж өгөх юм. Бүхэл бүтэн мэдээлэл Telegram бот дээр илгээгддэг.

## 🎮 Шинж Чанарууд

- ✅ Утасны дугаар оруулалт
- ✅ Алтын хэмжээ оруулалт (туухай хамгийн бага 100)
- ✅ MonPay QR код зургийг сонгох
- ✅ Данс шилжүүлэлтийн мэдээлэл (цаг, дүн, эзэмшигчийн нэр)
- ✅ GRAFFITI зургийн сонгох
- ✅ Telegram бот рүү автоматаар илгээлгэх
- ✅ Мобайл дээр сайн ажилладаг (Responsive)

## 📋 Шаардлагатай Зүйлс

- Node.js (v14 эсвэл түүнээс дээш)
- npm

## 🚀 Суулгах Алхамууд

### 1. Хамрааралтай сан клончлогдох эсвэл файлыг өргөлгөлгүүлэх

```bash
cd gamestore-website
```

### 2. Хамрааралтай модулиудыг суулгох

```bash
npm install
```

Энэ нь дараах сан судлуулч болох:
- express - веб сервер
- multer - файл сонгох
- axios - HTTP запрос
- dotenv - .env хэрэглэх

### 3. .env файлийг шалгах

`.env` файлийн доторь дараах мэдээлэл байх ёстой:

```
TELEGRAM_BOT_TOKEN=8697174872:AAE69AWBSkS1Ap3_Gmhu5TbXVc7f4AWvMws
TELEGRAM_CHAT_ID=7241705636
PORT=3000
```

**Сайн байна** - Энэ тохиргооны мэдээлэл аль хэдийн оруулсан у!

### 4. Вебсайтыг асаах

```bash
npm start
```

Эсвэл хөгжүүлэлтийн түүхэнүүдэд:

```bash
npm run dev
```

### 5. Веб сайтыг нээх

Браузерээ нээгээд энэ хаяг руу орно уу:

```
http://localhost:3000
```

## 📝 Өгөгдлийн Урсгал

1. Үйлчүүлэгч маягт бүрдүүлнэ
2. Үйлчүүлэгч "Болсон" товчийг дарна
3. Сервер мэдээлэл баталгаажуулна
4. Телеграм бот руу мэдээлэл илгээнэ
5. Админ Телеграм руу мэдээлэлийг хүлнэ

## 🐛 Асуудлын Сайгуул

### Портын дарга 3000 аль хэдийнэ ашиглагдаж байна
Өөр порт рүү өөрчилнө үү:
```bash
PORT=5000 npm start
```

### Telegram бот хариу алга
- Telegram Bot token болон Chat ID зүүлэлтийг шалгана уу
- Bot аль хэдийнэ сайтанд холбогдож байгаа эсэхийг нэмж байгаа уу

### Зургийн сонголт процествох
- Файлын хэмжээ тэнэмүүлэх боломжтойг бүр нэмэлтээр шалгана уу
- `server.js` рүү `fileSize` сонголтыг нэмж болов:

```js
const upload = multer({
  dest: 'uploads/',
  limits: { fileSize: 10 * 1024 * 1024 }, // 10MB хүртэл
  fileFilter: (req, file, cb) => {
    ...
  }
});
```
